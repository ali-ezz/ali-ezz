### 1. CareerBuddy — AI Career Intelligence Platform

CareerBuddy is an AI-powered job discovery platform that analyzes live job listings and estimates AI disruption risk using LLM-based reasoning.

```yaml
type: Applied AI product
status: deployed prototype
data_source: Remotive API
ai_layer: Groq / LLaMA
deployment: Vercel
main_problem: helping users understand career direction and job-market risk
```

**Core work**

- built live job-data ingestion from an external API
- integrated Groq / LLaMA for job-risk scoring
- created AI-powered career guidance features
- added filters for location, salary, remote work, and experience level
- deployed and iterated through Vercel

**Stack:** `JavaScript` · `Node.js` · `Vercel Serverless` · `Groq API` · `LLaMA` · `Remotive API`

**Repository:** [CareerBuddy](https://github.com/ali-ezz/CareerBuddy)
